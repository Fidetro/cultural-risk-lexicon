# 贡献指南

本项目是人工复核辅助工具，不是自动审核规则。贡献应当提升来源质量、降低误伤，或补充有清晰依据的别名和近似写法。

## 条目要求

- 每个条目至少包含一个稳定公开来源链接。
- 使用中性、面向复核的表述，避免直接写成违规判定。
- 区分规范名称、别名、拼音、数字形态和推测性近似写法。
- 不确定的写法应放入低置信别名、`fuzzy_aliases` 或测试用例，不要写成规范名称。
- 不接受仅基于社交媒体截图或传言新增的条目。

## 推荐 PR 形态

- 修改 `data/entities.json` 或 `data/events.json`。
- 在 `tests/test_query.py` 中增加或更新回归测试。
- 运行：

```bash
python3 -m unittest discover -s tests -v
```

## 数据复核说明

拼音类别名可同时包含空格形式和紧凑形式，例如 `yan song` 与 `yansong`。日期类别名可包含 ISO 日期、中文日期写法和常见数字写法，例如 `1937-12-13`、`1937年12月13日`、`12.13`。

来源优先级建议：

- 官方纪念馆、博物馆、国家公祭网、国家档案馆/档案局。
- 全国人大、中国政府网、人民网、新华网等稳定中文来源。
- 审判文书、档案汇编、学术资料。
- 权威百科和英文来源仅作为补充，不应长期替代中文权威来源。
